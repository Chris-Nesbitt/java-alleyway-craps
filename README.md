# Craps Game (Java GUI)

## Description
A graphical implementation of the casino dice game Craps developed in Java.
The application allows users to simulate the gameplay of Craps through an
interactive graphical interface.

The program implements the core rules of the game, including dice rolls,
game state management, and win/loss conditions. User interaction is handled
through GUI components and event-driven programming.

## Features
- Graphical user interface for interactive gameplay
- Randomized dice roll simulation
- Implementation of core Craps rules and win conditions
- Game state tracking and user feedback
- Event-driven input handling

## Purpose
This project was developed as part of coursework in the Computer Programming
and Analysis program at Fanshawe College.
